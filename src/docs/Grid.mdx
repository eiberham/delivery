# Grid


As its name states it's a grid that renders data.


## Properties


Prop | Type | Description
---|---|---
`field` | string | Name of the column header.
`label` | string | Identifier of the column.
`sort` | string | Sort order 'asc' or 'desc'.
`sortable` | boolean |  Indicates whether the cell header is sortable or not, later on it gets passed to the Cell component.
`header` | boolean | Indicates whether the cell is part of the header or not.
`handleSortClick` | function | Callback function to handle the sort method.
`key` | string | The cell key.
`shown` | boolean | Indicates whether the cell is visible or not.
