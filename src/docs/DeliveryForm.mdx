# DeliveryForm


Dynamic component used form creation and update of deliveries respectively.


## Properties

Prop | Type | Description
---|---|---
`mode` | string | Indicates whether it's the create or the update form.
`onSbmitForm` | callback | Callback function to be executed when form is submitted.
`onCancel` | callback | Callback function to be excuted when the cancel button is pressed.
`data` | array |  Array of objects used to fill every form input *(update mode)*.
